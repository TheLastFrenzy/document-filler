# 验收文档自动填充技能 (Document Filler)

## 用途

根据《台账清单》(.xlsx) 中指定服务目录的筛选数据，填充指定材料类型对应的Word验收文档模板或Excel说明材料模板。

支持的**材料类型**：
- `01-数据报表_需求文档` — 需求文档（需求来源表格 + 需求内容 + 内嵌报表清单）
- `01-数据统计分析_需求文档` — 数据统计分析需求文档（需求来源表格 + 需求内容 + 统计分析结果表清单 + 业务逻辑说明）
- `02-数据报表_设计文档` — 设计文档（业务分析表格 + 数据内容截图 + 数据来源库表清单 + 数据处理逻辑 + 报表指标设计）
- `02-数据统计分析_设计文档` — 数据统计分析设计文档（需求来源程序列表 + 实体表清单 + 结果表结构 + 数据处理流程图 + 数据加工逻辑）
- `03-数据报表_上线记录` — 上线记录（需求来源表格 + 产出说明 + 上线交付截图 + 使用记录截图）
- `03-数据统计分析_测试文档` — PDF测试文档（保留模板封面/修订记录/静态章节，重建目录跳转，按统计分析结果表拆分测试内容 + 解析工单自测报告内嵌截图 + 原版测试结论总数表）
- `04-数据统计分析_结果表及使用说明` — Excel说明材料（数据源表list + 表融合关系 + 结果表list + 结果表详情）

**材料类型由用户在指令中直接指定，不在台账清单中维护。**

## 使用方式

### 01 需求文档
```
使用 document-filler 技能，帮我生成服务目录为"N08-数据报表服务"的"01-数据报表_需求文档"文档材料，
台账清单的路径在 XXX，模版文件的路径在 XXX，输出的文件放在 XXX 路径下
```

### 01-数据统计分析 需求文档
```
使用 document-filler 技能，帮我生成服务目录为"N08-数据统计分析"的"01-数据统计分析_需求文档"文档材料，
台账清单的路径在 XXX，模版文件的路径在 XXX，输出的文件放在 XXX 路径下
```
默认会在模板同目录读取 `04-数据统计分析_结果表及使用说明.xlsx` 的 `2、表融合关系` sheet，将匹配结果表英文名的 `文字描述` 转为业务逻辑说明。

### 02 设计文档

```
台账清单的路径在 XXX，模版文件的路径在 XXX，参考的数据目录数据路径是 XXX，输出的文件放在 XXX 路径下
```

### 02-数据统计分析 设计文档
```
使用 document-filler 技能，帮我生成服务目录为"N08-数据统计分析"的"02-数据统计分析_设计文档"文档材料，
台账清单的路径在 XXX，模版文件的路径在 XXX，参考的数据目录数据路径是 XXX，输出的文件放在 XXX 路径下
```
默认会在模板同目录读取 `04-数据统计分析_结果表及使用说明.xlsx`，用于补齐源表、结果表字段、数据处理流程图和数据加工逻辑。该材料只生成 Word `.docx`，不会额外生成 PDF。

### 03 上线记录
```
使用 document-filler 技能，帮我生成服务目录为"N08-数据报表服务"的"03-数据报表_上线记录"文档材料，
台账清单的路径在 XXX，模版文件的路径在 XXX，输出的文件放在 XXX 路径下
```

### 03-数据统计分析 测试文档
 ```
使用 document-filler 技能，帮我生成服务目录为"N08-数据统计分析"的"03-数据统计分析_测试文档"文档材料，
台账清单的路径在 XXX，模版文件的路径在 XXX，输出的文件放在 XXX 路径下
```
台账需包含 `03-数据统计分析_测试文档_工单自测报告附件` 列。该列中每个工单内嵌的自测报告应为 Word 文档包，生成器会解析“4.检查和测试内容”章节，按统计分析结果表英文名提取七类截图并生成 PDF。最终 PDF 会保留模板的封面、修订记录和 1-4 章静态内容，只重建目录页、5.测试内容和 6.测试结论，并为目录写入可点击跳转。

### 04-数据统计分析 结果表及使用说明

```
台账清单的路径在 XXX，模版文件的路径在 XXX，参考的数据目录数据路径是 XXX，输出的文件放在 XXX 路径下
```

### 必须参数

| 参数 | 说明 | 适用材料类型 |
|------|------|-------------|
| **服务目录** | 台账清单中"服务目录"列的值 | 全部 |
| **材料类型** | 模板文件名（不含 .docx） | 全部 |
| **台账清单路径** | 台账清单 Excel 文件完整路径 | 全部 |
| **模板文件路径** | Word 模板(.docx) 或 Excel 模板(.xlsx) 完整路径 | 全部 |
| **输出文件路径** | 生成的 Word、PDF 或 Excel 说明材料保存路径；用户给文件夹时按材料类型生成文件名 | 全部 |
| **数据目录数据路径** | 数据目录数据 Excel 文件路径 | 02、04 |
| **台账清单路径**（03用） | 台账清单 Excel（含截图列） | 仅 03 |

## 执行流程

1. **读取台账清单**：打开 Excel，动态读取所有列
2. **筛选数据**：仅按 `服务目录` 列筛选匹配的行
3. **按材料类型分发**：根据 `material_type` 执行对应的填充逻辑
4. **提取图片**（03/04）：03数据报表从台账清单 cellimages.xml 提取上线截图和使用记录截图；03数据统计分析从工单自测报告附件提取测试截图；04为表融合关系生成血缘流程图并内嵌
4. **打开模板**：用户指定的模板文件路径
5. **填充内容**：按填充规则生成各章节内容
6. **更新目录 (TOC)**：Word材料通过 PowerShell COM 调用 Word 自动更新；Excel材料执行工作簿有效性与图片嵌入检查

核心脚本：`scripts/fill_document.py`

```bash
python scripts/fill_document.py \
  --service-dir "N08-数据报表服务" \
  --material-type "01-数据报表_需求文档" \
  --excel "台账清单路径.xlsx" \
  --template "模板文件路径.docx" \
  --output "输出文件路径.docx"

# 01-数据统计分析_需求文档
```
